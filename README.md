# Indie Website

A small indie-style website built using pure HTML, CSS, and JS featuring a retro terminal aesthetic inspired by The Matrix 
with a dark theme and neon green accents.

## Features
- Terminal-inspired UI design
- Multi-page layout (Home, About, Projects, Contact)
- Clean flexbox-based structure
- Easy-to-edit CSS variables for theming
- Lightweight and fast

## Tech Stack
- HTML5
- CSS3 (Flexbox, custom properties)

## Project Structure
- index.html – Home page  
- styles.css – Main stylesheet
- scripts.js - JavaScript 
- pages/ – Additional pages  
- assets/ – Images and media files  

## Purpose
A personal indie web project focused on experimenting with layout, design, and creating a unique and fun retro terminal vibe for learning and creativity.
